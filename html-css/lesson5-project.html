3. ملف CSS الكامل - بنفس ستايل موقعك

اعمل ملف style.css جنب ملف HTML وحط فيه:

* { margin: 0; padding: 0; box-sizing: border-box; }
body { font-family: 'Cairo', sans-serif; background: #000; color: #fff; }

/* Navbar */

  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 20px 5%;
  background: rgba(0, 0, 0, 0.8);
  backdrop-filter: blur(10px);
  border-bottom: 2px solid #00f5ff;
  position: sticky;
  top: 0;
}

4. Checklist قبل ما ترفع المشروع

  • حطيت Meta Viewport في head
  • كل الصور عندها alt
  • جربت الموقع على الموبايل F12
  • الألوان واضحة والخط مقروء
  • الزرار شغال بيودي للقسم الصح
  • رفعت الملفات على GitHub Pages
