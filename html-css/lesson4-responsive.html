الدرس 4: التصميم المتجاوب Responsive

المدة: 20 دقيقة | المستوى: مهم جداً

1. يعني إيه Responsive؟

يعني موقعك يشتغل ويبقى شكله حلو على الكمبيوتر + التابلت + الموبايل
من غير ما المستخدم يعمل Zoom أو يتحرك يمين وشمال.

إحصائية: 70% من الناس بتفتح من الموبايل. لو موقعك مش Responsive هتخسرهم كلهم.

2. القاعدة الذهبية: Meta Viewport

أول سطر لازم تحطه في أي صفحة HTML عشان الموبايل يفهمها:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
من غير السطر ده الموبايل هيفتح موقعك كأنه شاشة كمبيوتر ويصغره جامد.

3. Media Query = السحر كله

كود بيقول للمتصفح: "لو الشاشة أصغر من 768px اعمل كذا"

/* للكمبيوتر - الشاشات الكبيرة */
.container { width: 1200px; }
h1 { font-size: 48px; }

/* للموبايل - الشاشات أقل من 768px */
@media (max-width: 768px) {
.container { width: 100%; padding: 0 15px; }
  h1 { font-size: 28px; } /* صغر الخط */
.lessons-grid { grid-template-columns: 1fr; } /* خلي الكروت تحت بعض */
}

4. أهم 4 قواعد للموبايل

❌ غلط: تستخدم px ثابت

.box { width: 400px; }

المشكلة: على موبايل شاشته 360px البوكس هيطلع بره

✅ صح: تستخدم % أو max-width

.box { width: 90%; max-width: 400px; }

الحل: هياخد 90% من الشاشة مهما كان حجمها

القواعد الأربعة:

1. الصور: img { max-width: 100%; height: auto; }
2. النصوص: استخدم rem أو em بدل px
3. Grid/Flex: grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
4. اخفي العناصر التقيلة: @media (max-width:768px) {.sidebar { display:none; } }

5. Breakpoints المشهورة

دي المقاسات اللي كل الناس بتستخدمها:

/* موبايل */
@media (max-width: 480px) {... }

/* تابلت */
@media (max-width: 768px) {... }

/* لابتوب صغير */
@media (max-width: 1024px) {... }

/* كمبيوتر */
@media (min-width: 1025px) {... }

6. مثال: كروت الكورسات بتاعتك

شوف الكود ده في html-css.html:

.lessons-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 20px;
}

@media (max-width: 768px) {
.course-title { flex-direction: column; text-align: center; }
.course-title h2 { font-size: 24px; }
}

الكود ده بيخلي الكروت جنب بعض على الكمبيوتر، ولو فتحت موبايل ينزلوا تحت بعض + العنوان يصغر

7. ازاي تجرب موقعك على الموبايل؟

1. افتح الموقع على Chrome
2. دوس F12 أو كليك يمين → Inspect
3. دوس على أيقونة الموبايل 📱 فوق
4. اختار iPhone أو Android وجرب

8. تحميل قالب Responsive جاهز

عملتلك صفحة HTML كاملة Responsive تاخدها كوبي بيست وتعدل عليها
