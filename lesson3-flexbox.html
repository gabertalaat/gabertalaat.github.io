الدرس 3: Flexbox & Grid

المدة: 25 دقيقة | المستوى: متوسط

1. المشكلة اللي بيحلوها

زمان كنا بنرتب العناصر بـ float و position وكان كابوس 😂
Flexbox & Grid جم خلونا نرتب أي حاجة في 3 سطور كود بس.

2. Flexbox = صف أو عمود مرن

استخدمه لما عايز ترتب عناصر جنب بعض أو فوق بعض

.container {
  display: flex; /* شغل الفليكس */
  justify-content: space-between; /* وزع العناصر بالتساوي */
  align-items: center; /* وسطنهم في النص عمودي */
  gap: 20px; /* مسافة 20px بين كل عنصر */
}

أهم 3 خصائص في Flexbox:
1. justify-content = ترتيب أفقي: start, center, end, space-between
2. align-items = ترتيب عمودي: start, center, end
3. flex-direction = الاتجاه: row أو column

3. Grid = جدول احترافي

استخدمه لما عايز تقسم الصفحة أعمدة وصفوف زي المجلة

.container {
  display: grid; /* شغل الجريد */
  grid-template-columns: 1fr 1fr 1fr; /* 3 أعمدة متساوية */
  gap: 15px; /* مسافة 15px بين الكروت */
}

1fr = جزء واحد من المساحة المتاحة. لو كتبت 1fr 2fr يبقى العمود التاني ضعف الأول

4. إمتى تستخدم Flexbox وإمتى Grid؟

Flexbox: للـ Navbar، الكروت اللي جنب بعض، توسيط عنصر
Grid: لتقسيم الصفحة كلها، معرض الصور، Dashboard كامل

قاعدة سهلة: لو بعد واحد = Flexbox، لو بعدين = Grid

5. مثال: كروت الكورسات بتاعتك

الكروت اللي في html-css.html معمولة بـ Grid كده:

.lessons-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 20px;
}

الكود ده بيخلي الكروت تترص جنب بعض، ولو الشاشة صغرت ينزلوا تحت بعض أوتوماتيك ✨
