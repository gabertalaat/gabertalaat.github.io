الدرس 4: API & Fetch

المدة: 30 دقيقة | المستوى: متوسط

1. يعني إيه API؟

API = Application Programming Interface
تخيل مطعم: انت العميل، المطبخ هو السيرفر، والجرسون هو الـ API.
انت بتطلب من الجرسون "عايز بيتزا"، هو بيروح المطبخ ويجيبهالك.

بالمصري: API = وسيط بيجيبلك بيانات من سيرفر تاني وانت في موقعك 😂

2. Fetch = الطريقة الحديثة تجيب بيانات

قبل 2015 كنا بنستخدم AJAX معقد. دلوقتي fetch() سهل جداً:

// الهيكل الأساسي
fetch('رابط الـ API')
 .then(response => response.json()) // حول الرد لـ JSON
 .then(data => {
    console.log(data); // البيانات وصلت خلاص
 })
 .catch(error => {
    console.log('حصل خطأ:', error);
 });
async/await = طريقة أسهل وأوضح. هنستخدمها في الأمثلة الجاية

3. مثال 1: هات نكتة عشوائية 🔥

هنستخدم API مجاني بيجيب نكت برمجة:

const getJoke = async () => {
  const response = await fetch('https://api.chucknorris.io/jokes/random');
  const data = await response.json();
  console.log(data.value); // النكتة هنا
}

4. مثال 2: طقس القاهرة مباشر

هنستخدم API الطقس. الكود ده هيجيب درجة حرارة القاهرة دلوقتي:

const getWeather = async () => {
  const url = 'https://api.open-meteo.com/v1/forecast?latitude=30.05&longitude=31.25¤t=temperature_2m';
  const response = await fetch(url);
  const data = await response.json();
  const temp = data.current.temperature_2m;
  console.log(`الحرارة في القاهرة: ${temp}°C`);
}

5. إرسال بيانات POST

لما تحب تبعت بيانات للسيرفر زي تسجيل دخول:

const sendData = async () => {
  const response = await fetch('https://api.example.com/login', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      username: "gaber",
      password: "12345"
    })
  });
  const data = await response.json();
}

6. التعامل مع الأخطاء try/catch

لازم تحمي كودك لو النت فصل أو الـ API وقع:

const getData = async () => {
  try {
    const response = await fetch('https://api.example.com/data');
    if (!response.ok) throw new Error('السيرفر مش شغال');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.log('حصل خطأ:', error.message);
    alert('النت فصل أو الـ API واقع 😢');
  }
}
