الدرس 1: أساسيات JS - المتغيرات والدوال

المدة: 20 دقيقة | المستوى: مبتدئ

1. يعني إيه JavaScript؟

JS = JavaScript
دي لغة "المخ" بتاع الموقع. بتخليه يتفاعل معاك.
HTML = الهيكل، CSS = الشكل، JS = الحركة والعقل

مثال: لما تدوس على زرار ويطلعلك رسالة... ده JS

2. أول كود JS في حياتك

حط الكود ده في أي صفحة HTML قبل قفلة </body>:

<script>
  alert("أهلاً بيك في عالم JS يا جابر 😂");
</script>

جرب دوس على الزرار ده:

3. المتغيرات: var vs let vs const

المتغير = علبة بتخزن فيها قيمة

var name = "جابر"; // قديم، متستخدموش
let age = 25; // حديث، تقدر تغير القيمة
const country = "مصر"; // ثابت، مينفعش تغيره
القاعدة: استخدم const دايماً، ولو هتغير القيمة استخدم let. انسا var خالص

4. أنواع البيانات الأساسية

let name = "جابر"; // String نص
let age = 25; // Number رقم
let isPro = true; // Boolean صح أو غلط
let skills = ["HTML", "CSS", "JS"]; // Array مصفوفة

5. أول Function ليك

الـ Function = مجموعة أوامر بتناديها وقت ما تحب

function sayHello(userName) {
  alert("أهلاً يا " + userName + " 🔥");
}

// تشغيل الدالة
sayHello("جابر");

6. Console.log = صاحبك الصدوق

عشان تشوف المتغيرات والقيم من غير alert:

let x = 5;
let y = 10;
console.log(x + y); // هيطبع 15 في Console
ازاي تفتح Console: دوس F12 في المتصفح → اختار Console
