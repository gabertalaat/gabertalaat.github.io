الدرس 5: مشروع لعبة "خمن الرقم"

المدة: 45 دقيقة | المستوى: مشروع تطبيقي

1. فكرة اللعبة

الكمبيوتر هيختار رقم عشوائي من 1 لـ 100
وانت تحاول تخمنه. كل مرة هيقولك "أعلى" أو "أقل"
لحد ما تجيبه صح. هنسجل أقل عدد محاولات جبتها 💙

2. اللعبة شغالة قدامك أهي 👇

خمن رقم من 1 لـ 100


المحاولات: 0

أفضل نتيجة: -

3. شرح الكود: إزاي اتعملت؟

1. المتغيرات الأساسية:

let secretNumber = Math.floor(Math.random() * 100) + 1;
let attempts = 0;
let highScore = localStorage.getItem('highScore') || 0;

2. دالة التخمين:

const checkGuess = () => {
  const guess = Number(document.getElementById('guessInput').value);
  attempts++;

  if (guess === secretNumber) {
    document.getElementById('message').textContent = 
      `صح! الرقم كان ${secretNumber} 🎉`;
    document.getElementById('message').style.color = '#00f5ff';
    
    if (attempts < highScore || highScore === 0) {
      highScore = attempts;
      localStorage.setItem('highScore', highScore);
    }
  } else if (guess > secretNumber) {
    document.getElementById('message').textContent = 'أقل شوية 👇';
    document.getElementById('message').style.color = '#ff4444';
  } else {
    document.getElementById('message').textContent = 'أعلى شوية 👆';
    document.getElementById('message').style.color = '#ff4444';
  }
  
  document.getElementById('attempts').textContent = `المحاولات: ${attempts}`;
}

3. دالة إعادة التشغيل:

const resetGame = () => {
  secretNumber = Math.floor(Math.random() * 100) + 1;
  attempts = 0;
  document.getElementById('message').textContent = '';
  document.getElementById('attempts').textContent = 'المحاولات: 0';
  document.getElementById('guessInput').value = '';
}

4. طور اللعبة بنفسك

  • ضيف مستويات صعوبة: سهل 1-50، صعب 1-500
  • ضيف مؤقت: لازم تجيب الرقم في 30 ثانية
  • ضيف ألوان: لو قربت اللون يبقى أزرق، لو بعيد أحمر
  • اعمل Leaderboard يحفظ أسماء اللاعبين
