الدرس 2: البيانات والـ Loops والـ Conditions

المدة: 35 دقيقة | المستوى: مبتدئ

1. أنواع البيانات الأساسية

بايثون فيها 4 أنواع هتستخدمهم كل يوم:

# 1. String - نص
name = "جابر"
message = 'بحب بايثون 🔥'

# 2. Integer - رقم صحيح
age = 20
users = 1000

# 3. Float - رقم عشري
price = 99.99
pi = 3.14

# 4. Boolean - صح ولا غلط
is_online = True
is_admin = False

2. Lists = مصفوفات يا معلم 📋

دي زي صندوق بتحط فيه كذا حاجة مع بعض

skills = ["HTML", "CSS", "JS", "Python"]
numbers = [10, 20, 30, 40]

# نطبع أول عنصر
print(skills[0]) # HTML

# نضيف عنصر جديد
skills.append("React")
print(skills) # ['HTML', 'CSS', 'JS', 'Python', 'React']

# عدد العناصر
print(len(skills)) # 5

3. Dictionaries = قواميس 🗂️

دي بقى بتخزن البيانات على شكل Key: Value

user = {
    "name": "جابر",
    "age": 20,
    "city": "Cairo",
    "skills": ["Python", "JS"]
}

print(user["name"]) # جابر
print(user["skills"][0]) # Python

# نضيف حاجة جديدة
user["email"] = "gaber@example.com"

4. الـ Conditions = لو حصل كذا اعمل كذا

age = 18

if age >= 18:
    print("انت بالغ يا معلم")
elif age >= 13:
    print("انت مراهق")
else:
    print("انت لسه صغير")

# النتيجة: انت بالغ يا معلم
ملاحظة مهمة: بايثون بتعتمد على المسافات ⚠️
لازم الكود اللي جوا الـ if يبقى مزاح 4 مسافات لجوا

5. الـ Loops = التكرار 🔄

For Loop - تلف على List:

skills = ["HTML", "CSS", "JS", "Python"]

for skill in skills:
    print(f"أنا بعرف {skill}")
أنا بعرف HTML
أنا بعرف CSS
أنا بعرف JS
أنا بعرف Python

While Loop - طول ما الشرط متحقق:

count = 1

while count <= 5:
    print(f"العد: {count}")
    count = count + 1
العد: 1
العد: 2
العد: 3
العد: 4
العد: 5

6. تمرين تطبيقي: برنامج إدارة مهام

tasks = []

# نضيف مهام
tasks.append("اذاكر Python")
tasks.append("اعمل مشروع")
tasks.append("ارفع على GitHub")

# نطبع كل المهام
print("مهامك النهاردة:")
for i, task in enumerate(tasks, 1):
    print(f"{i}. {task}")
