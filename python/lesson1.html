الدرس 1: مقدمة في بايثون والتنصيب

المدة: 20 دقيقة | المستوى: مبتدئ

1. ليه بايثون أصلاً؟ 🐍

بايثون أسهل لغة برمجة ممكن تبدأ بيها. بتكتب كود قليل وتعمل حاجات كتير.
بتستخدم في: مواقع، ذكاء اصطناعي، بوتات، ألعاب، تحليل بيانات... كل حاجة حرفياً 😂

2. تنصيب Python على جهازك

Windows:

1. ادخل على python.org
2. دوس Download Python 3.12
3. وانت بتنصب علم على "Add Python to PATH" ⚠️ مهم جداً
4. دوس Install Now

نتأكد إنه اتنصب صح:

# افتح CMD أو Terminal واكتب
python --version

# المفروض يطلعلك
Python 3.12.0

3. أول كود ليك

اعمل ملف جديد اسمه app.py واكتب فيه:

print("أهلا يا جابر 🔥")
print("أنا بكتب بايثون لأول مرة")

شغله من الـ Terminal:

python app.py
VS Code = أحسن محرر تكتب فيه بايثون
نزل إضافة Python من Microsoft وهتبقى تمام

4. المتغيرات في بايثون

في بايثون مش محتاج تكتب `let` أو `const` زي JS 😂

name = "جابر"
age = 20
is_developer = True

print(name) # جابر
print(age) # 20
