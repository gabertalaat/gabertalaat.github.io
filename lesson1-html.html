الدرس 1: مقدمة في HTML وبناء الهيكل

المدة: 15 دقيقة | المستوى: مبتدئ جداً

1. يعني إيه HTML؟

HTML = HyperText Markup Language
دي لغة "الهيكل" بتاع أي موقع. زي الطوب في البيت بالظبط.
من غيرها مفيش موقع... فيسبوك ويوتيوب كله HTML في الأساس.

مهم: HTML مش لغة برمجة. دي لغة وصف بتقول للمتصفح "ده عنوان" و"ده نص" و"ده صورة".

2. أهم 5 تاجات هتستخدمهم كل يوم

<h1>عنوان كبير جداً</h1> ← للعنوان الرئيسي
<h2>عنوان أصغر</h2> ← للعناوين الفرعية
<p>دي فقرة نص عادي</p> ← لأي كلام
<img src="صورة.jpg" alt="وصف"> ← عشان تحط صورة
<a href="google.com">دوس هنا</a> ← عشان تعمل لينك

3. هيكل أي صفحة HTML

احفظ الشكل ده. أي صفحة في الكون لازم تبقى كده:

<!DOCTYPE html>
<html>
  <head>
    <title>عنوان الصفحة</title>
  </head>
  <body>
    <h1>أهلاً بالعالم!</h1>
    <p>دي أول صفحة ويب أعملها</p>
  </body>
</html>
<head> = دماغ الصفحة. حاجات مش بتظهر زي العنوان
<body> = جسم الصفحة. كل اللي بتشوفه بعينك

4. جرب بنفسك دلوقتي ⚡

1. افتح Notepad
2. انسخ الكود اللي فوق
3. احفظه باسم index.html
4. افتحه بالمتصفح... مبروك عملت أول موقع 🎉
