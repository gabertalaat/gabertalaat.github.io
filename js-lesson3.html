الدرس 3: ES6+ الميزات الحديثة

المدة: 30 دقيقة | المستوى: مهم جداً

1. يعني إيه ES6؟

ES6 = ECMAScript 2015
ده تحديث ضخم نزل للـ JS سنة 2015 غير اللغة 180 درجة.
بالمصري: قبل ES6 كنت بتكتب كود كتير ومتعب، بعده بقى الكود قصير ونضيف 😂

2. Arrow Functions => السهم السحري

بدل ما تكتب function طويلة، اكتبها في سطر:

❌ الطريقة القديمة: 
function sum(a, b) {
  return a + b;
}
✅ الطريقة الجديدة ES6: 
const sum = (a, b) => a + b;

جرب دوس وشوف النتيجة:

القاعدة: لو الدالة سطر واحد، شيل {} و return واكتبها علطول

3. Template Literals = الكتابة الحديثة

انسا + عشان تلزق نصوص. استخدم ` `:

const name = "جابر";
const age = 25;

// قديم
let msg = "أهلاً يا " + name + " عمرك " + age;

// جديد ES6
let msg = `أهلاً يا ${name} عمرك ${age} 🔥`;

4. Destructuring = فك العلبة بسرعة

بدل ما تكتب user.name و user.age كل مرة:

const user = { name: "جابر", age: 25, city: "القاهرة" };

// قديم
let name = user.name;
let age = user.age;

// جديد ES6 - فكها في سطر
const { name, age, city } = user;
console.log(name); // جابر

نفس الكلام مع Arrays:

const colors = ["أحمر", "أخضر", "أصفر"];
const [first, second] = colors;
console.log(first); // أحمر

5. Spread Operator... = انسخ ووزع

الـ ... دي بتعمل سحر:

// 1. انسخ Array
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5]; // [1,2,3,4,5]

// 2. ادمج Objects
const user = { name: "جابر" };
const userWithAge = {...user, age: 25 };
// { name: "جابر", age: 25 }

// 3. ابعت كل عناصر Array كـ parameters
const nums = [5, 10, 15];
console.log(Math.max(...nums)); // 15

6. Default Parameters = قيم افتراضية

// لو المستخدم مبعتش قيمة، خد الافتراضي
const greet = (name = "يا غريب") => {
  return `أهلاً ${name}`;
}

greet("جابر"); // أهلاً جابر
greet(); // أهلاً يا غريب

7. let و const بدل var

الخلاصة:
1. const = ثابت، استخدمه 90% من الوقت
2. let = متغير، لو هتغير القيمة
3. var = انساه خالص، ده بتاع 2010 😂
