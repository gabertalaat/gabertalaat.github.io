الدرس 2: أساسيات CSS والتنسيق

المدة: 20 دقيقة | المستوى: مبتدئ

1. يعني إيه CSS؟

CSS = Cascading Style Sheets
دي لغة "الديكور والألوان". لو HTML هو الطوب، CSS هو الدهان والعفش والإضاءة.
بتتحكم في: الألوان، الخطوط، الأحجام، المسافات، الحركات.

القاعدة: HTML = الهيكل، CSS = الشكل الفاجر

2. 3 طرق تربط CSS بـ HTML

1. مباشر جوه التاج:
<h1 style="color:red">عنوان أحمر</h1>

2. في نفس الصفحة:
<style>
  h1 { color: red; }
</style>

3. ملف خارجي - الأفضل:
<link rel="stylesheet" href="style.css">

3. أهم 6 خصائص CSS هتستخدمهم كل يوم

color: #00FF88; /* لون النص - أخضر ماتريكس */
background: #000000; /* لون الخلفية - أسود */
font-size: 24px; /* حجم الخط */
font-weight: bold; /* تخن الخط */
text-align: center; /* وسّط النص */
margin: 20px; /* مسافة بره العنصر */

4. مثال عملي: نلون صفحة الدرس الأول

فاكر كود HTML من الدرس الأول؟ يلا نلونه:

<style>
  body {
    background: #0a0a0a; /* خلفية سودة */
    color: #fff; /* النص أبيض */
  }
  h1 {
    color: #00FF88; /* العنوان أخضر */
    font-size: 48px; /* خط كبير */
    text-align: center; /* في النص */
  }
  p {
    color: #ddd; /* النص رمادي فاتح */
    line-height: 1.8; /* مسافة بين السطور */
  }
</style>

النتيجة هتبقى كده

نص لونه رمادي وشكله نضيف

5. الـ Class والـ ID

عشان تستهدف عنصر معين وتديله ستايل لوحده:

<!-- في HTML -->
<p class="text-gold">نص دهبي</p>
<p id="special">نص مميز</p>

<!-- في CSS -->
.text-gold { color: #FFD700; }
#special { color: red; font-size: 30px; }
Class =. بتستخدمه مع كذا عنصر
ID = # بتستخدمه مع عنصر واحد بس

6. تحميل ملف style.css جاهز

عملتلك ملف CSS فيه كل الألوان والستايل بتاع الماتريكس. حمله واربطه بأي صفحة HTML

تحميل style.css 📥
